# New York Oil & Gas Production Analytics Project

## 📌 Project Overview
This is an end-to-end data analytics project focused on analyzing the historical oil and gas production trend in New York. The goal is to clean production data, extract deep analytical insights using SQL, and build a live interactive dashboard for executive decision-making.

## 🛠️ Tech Stack Used
- **Python**: For Data Cleaning, Handling Missing Values, and ETL.
- **MySQL**: For Advanced Data Querying (Window Functions, Aggregate Analytics).
- **Tableau**: For Professional Dashboard Design and Geospatial Mapping.
